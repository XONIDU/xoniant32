REQUISITOS PARA INSTALAR XONIANT32 ULTIMATE
============================================

1. REQUISITOS DE HARDWARE
-------------------------
- Procesador: 32 bits (i386) compatible (Intel Pentium III / Celeron o superior)
- Memoria RAM: 512 MB mínimo (1 GB recomendado)
- Almacenamiento: 8 GB libres en el disco de destino
- Gráficos: Cualquier chip compatible con Xorg (controladores fbdev/vesa incluidos)
- Hardware probado: ASUS Eee PC 900 (Intel Celeron M 900MHz, 1GB RAM, GMA 900)

2. REQUISITOS DE SOFTWARE BASE
------------------------------
- Tener instalado antiX Linux 32 bits (versión 23.2 o superior recomendada)
- antiX debe estar funcionando correctamente (puede ser instalado en disco o en live USB)
- Conexión a internet activa (cable Ethernet o WiFi configurado)

3. REQUISITOS DE RED
--------------------
- Conexión a internet estable (para descargar paquetes y herramientas XONI)
- Para WiFi: connman viene incluido en antiX y se usará durante la instalación

4. ARCHIVOS NECESARIOS
----------------------
- Script de instalación: install-xoniant32-ultimate.sh
- Tamaño del script: ~35 KB
- El script puede descargarse desde: https://github.com/XONIDU/xoniant32

5. PERMISOS Y EJECUCIÓN
------------------------
- El script debe ejecutarse con privilegios de superusuario (sudo)
- Dar permisos de ejecución: chmod +x install-xoniant32-ultimate.sh
- Ejecutar: sudo ./install-xoniant32-ultimate.sh

6. TIEMPO ESTIMADO
-------------------
- Instalación completa: 15-30 minutos (dependiendo de la velocidad de internet)
- La ISO resultante (si se crea) ocupará aproximadamente 1 GB

7. NOTAS IMPORTANTES
--------------------
- El script eliminará automáticamente paquetes innecesarios (CUPS, Bluetooth, etc.)
- Conserva todos los controladores de video, audio y red
- Después de la instalación, el sistema arrancará directamente en terminal fija
- No se requiere intervención del usuario durante el proceso principal
- Se recomienda hacer una copia de seguridad de datos importantes antes de continuar

8. SOPORTE Y ACTUALIZACIONES
-----------------------------
- Repositorio oficial: https://github.com/XONIDU/xoniant32
- Para actualizar: sudo xoni-update
- Reportar problemas: xonidu@gmail.com
